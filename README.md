# Marcosoft Chat Counter

This app records all the chats I receive each day and then calculates the percentage of each type as follows:

1. Simple Questions
2. Boss
3. Difficult Questions
4. No Action

The idea behind it is to validate if I, indeed, get a high amount of difficult questions per day.
